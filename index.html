Module 1

Getting started with EndNote

Learn what EndNote is, what a library does, and how to create your first library file.

Practice task: Open EndNote 2025 and create a new library named MyResearch.enl.
Show guided steps
  1. Open EndNote.
  2. Select Create a New Library.
  3. Name it MyResearch.enl.
  4. Save it somewhere easy to find.

Module 2

Adding references

Learn three beginner-friendly ways to add content: manual entry, importing a file, and bringing in a PDF.

Practice task: Add one journal article, one book, and one PDF-based reference.
Show guided steps
  1. Use References > New Reference for manual entry.
  2. Use the import option for exported database files.
  3. Drag and drop a PDF to test automatic reference creation.

Quick check

Module 4

Cite While You Write in Word

Learn how to insert citations into Microsoft Word and switch between reference styles like APA 7th.

Practice task: Insert one citation in Word and change the bibliography style to APA 7th.
Show guided steps
  1. Open a Word document.
  2. Go to the EndNote tab in Word.
  3. Choose Insert Citation.
  4. Select a style such as APA 7th.
