EndNote 2025 Interactive Starter Tutorial
This is a lightweight, GitHub-ready static prototype for an interactive beginner tutorial on EndNote 2025.
What is included
`index.html` - the tutorial page
`style.css` - styling
`script.js` - progress tracking and quiz logic
Features
Beginner-focused modules
Practice tasks for each module
Mini quizzes for quick self-checks
Progress saved in browser local storage
Official EndNote resource links
How to test locally
Download or clone the project.
Open `index.html` in a browser.
Work through the modules and test progress tracking.
Good next upgrades
Add screenshots or short screen recordings
Add branching paths for Windows vs macOS
Add a certificate of completion
Add admin analytics if later hosted on a server
Replace generic tasks with UFS-specific EndNote workflows
GitHub Pages
This prototype can be pushed straight to a GitHub repository and hosted with GitHub Pages.
